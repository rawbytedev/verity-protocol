# Verity Protocol Compliance Suite

## Overview

This directory contains test suites and validation tools for verifying compliance with the Verity Protocol specification.

## Test Categories

### 1. Schema Validation

- Verity Claim JSON Schema validation
- DID Document structure validation
- Proof format validation

### 2. Cryptographic Compliance

- Ed25519 signature generation/verification
- SHA-256 hashing correctness
- Key derivation compliance

### 3. API Compliance

- Ledger Gateway API endpoints
- Verification API responses
- Error handling consistency

### 4. Security Compliance

- Key management procedures
- Revocation handling
- Privacy requirements
